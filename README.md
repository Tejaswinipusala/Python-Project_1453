# Amazon Products Sales Analysis

A simple exploratory data analysis (EDA) project on Amazon product sales data.

---

## Dataset
Dataset used: [Amazon Products Sales Data (Cleaned) on Kaggle](https://www.kaggle.com/datasets/)

---

## What it does
- Cleans missing values for ratings, discounted price, and discount percentage.
- Visualizes:
  - Product ratings distribution
  - Top 10 categories by count
  - Top 10 categories by average discount %
  - Discount % vs rating (scatter)

---

## How to run
1. Clone this repo and download the dataset from Kaggle.
2. Install dependencies:
```bash
pip install pandas numpy matplotlib
````

3. Run the script:

```bash
python analysis.py
```
